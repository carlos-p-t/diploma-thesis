### About the partially functional implementation

The directory [**Object Detector**](https://github.com/carlos-p-t/diploma-thesis/tree/main/Object%20Detector) contains the files used to train and validate the object detector
of this project. There are three main important files: The [data_processing](https://github.com/carlos-p-t/diploma-thesis/blob/main/Object%20Detector/data_preprocessing.ipynb) 
notebook contains the code that was used to parse and preprocess the data for training. The output of this notebook were the .txt files. The [training](https://github.com/carlos-p-t/diploma-thesis/blob/main/Object%20Detector/training.ipynb)
notebook contains the code used to train the models online (through Google Cloud services). The output were the files located in the *results/detect* folder. Lastly, the
[results](https://github.com/carlos-p-t/diploma-thesis/blob/main/Object%20Detector/results.ipynb) notebook contains the visualization of the results obtained after training.
To run the visualization, the notebook can be downloaded along with the *results/detect* and *sample test images* folders.
